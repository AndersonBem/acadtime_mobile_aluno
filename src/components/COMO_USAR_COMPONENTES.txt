COMO USAR OS COMPONENTES
========================

Esta pasta tem componentes reutilizaveis do app mobile.
Eles servem para evitar repetir o mesmo visual em varias telas.

Para usar qualquer componente:
1. Importe no topo da pagina.
2. Chame o componente dentro do return.


1. AppHeader
------------
Cabecalho do app com logo, sino e perfil.
O sino e o perfil podem navegar para outras telas.
Se receber photoUrl, mostra a foto do aluno. Se nao receber, mostra o icone padrao.

Import:
import AppHeader from '../components/AppHeader';

Uso:
<AppHeader
  photoUrl={dashboard.aluno?.foto_perfil_url}
  onPressNotifications={() => navigation.navigate('Notifications')}
  onPressProfile={() => navigation.navigate('Profile')}
/>

Props:
- photoUrl: URL da foto do aluno. Opcional.
- onPressNotifications: funcao chamada ao tocar no sino.
- onPressProfile: funcao chamada ao tocar no perfil.


2. BottomTabBar
---------------
Menu inferior com icones de home, submissao, adicionar, notificacoes e perfil.
Ele ja esta preparado para navegacao.

Import:
import BottomTabBar from '../components/BottomTabBar';

Uso:
<BottomTabBar
  activeRoute="Dashboard"
  onNavigate={(screen) => navigation.navigate(screen)}
/>

Props:
- activeRoute: nome da rota ativa. Exemplo: Dashboard, Submissions, NewSubmission, Notifications, Profile.
- onNavigate: funcao chamada quando o usuario toca em um item do menu.

Rotas usadas pelo BottomTabBar:
- Dashboard
- Submissions
- NewSubmission
- Notifications
- Profile

Importante:
Essas rotas precisam existir no Routes.js para que navigation.navigate(screen) funcione.


3. PageTitle
------------
Titulo da pagina. Pode ter seta de voltar.

Import:
import PageTitle from '../components/PageTitle';

Uso simples:
<PageTitle title="Minhas submissoes" />

Uso com voltar:
<PageTitle
  title="Detalhes da submissao"
  showBack={true}
  onBack={() => console.log('Voltar')}
/>

7. InfoCard
-----------
Card branco com sombra para agrupar informacoes.
Tudo que ficar dentro dele aparece dentro do card.

Import:
import InfoCard from '../components/InfoCard';

Uso:
<InfoCard>
  <Text>Curso</Text>
  <Text>Analise e Desenvolvimento de Sistemas</Text>
</InfoCard>

Observacao:
Se usar Text dentro da pagina, lembre de importar:
import { Text } from 'react-native';


8. InputField
-------------
Campo de texto para e-mail, senha, busca, matricula etc.

Import:
import InputField from '../components/InputField';

Uso com estado:
const [email, setEmail] = useState('');

<InputField
  label="E-mail"
  placeholder="Digite seu e-mail"
  value={email}
  onChangeText={setEmail}
  icon="mail"
/>

Uso para senha:
<InputField
  label="Senha"
  placeholder="Digite sua senha"
  value={senha}
  onChangeText={setSenha}
  icon="lock-closed"
  secureTextEntry={true}
/>

Observacao:
Se usar useState, importe:
import { useState } from 'react';

11. ProgressRow
---------------
Linha de progresso usada no Dashboard para mostrar horas por tipo de atividade.

Import:
import ProgressRow from '../components/ProgressRow';

Uso:
<ProgressRow
  icon="document-text-outline"
  title="Curso de Extensao"
  current={10}
  total={20}
  percent={50}
  color={colors.primary}
  backgroundColor="#e8f1fb"
/>

Observacao:
O percent deve ficar entre 0 e 100 para a barra nao ultrapassar o limite.


Exemplo de tela usando varios componentes
-----------------------------------------

import { SafeAreaView, StyleSheet, View } from 'react-native';
import AppHeader from '../components/AppHeader';
import BottomTabBar from '../components/BottomTabBar';
import PageTitle from '../components/PageTitle';
import InfoCard from '../components/InfoCard';
import PrimaryButton from '../components/PrimaryButton';
import { colors, spacing } from '../styles/global';

export default function MinhaPagina() {
  return (
    <SafeAreaView style={styles.screen}>
      <AppHeader
        onPressNotifications={() => navigation.navigate('Notifications')}
        onPressProfile={() => navigation.navigate('Profile')}
      />

      <View style={styles.content}>
        <PageTitle title="Minha pagina" />

        <InfoCard>
          <PrimaryButton
            title="Clique aqui"
            onPress={() => console.log('Clicou')}
          />
        </InfoCard>
      </View>

      <BottomTabBar
        activeRoute="Dashboard"
        onNavigate={(screen) => navigation.navigate(screen)}
      />
    </SafeAreaView>
  );
}

const styles = StyleSheet.create({
  screen: {
    flex: 1,
    backgroundColor: colors.background,
  },

  content: {
    flex: 1,
    padding: spacing.lg,
  },
});
